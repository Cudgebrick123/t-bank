# 📚 T-Библиотека

Консольное приложение для управления личной библиотекой книг с возможностью сохранения данных в файл.

## ✨ Возможности

- **Добавление книг** - добавление новых книг с указанием названия, автора, жанра, года издания и описания
- **Просмотр книг** - просмотр всех книг с сортировкой (по названию, автору, году, жанру) и фильтрацией (по жанру, статусу)
- **Избранное** - добавление книг в избранное для быстрого доступа
- **Статус чтения** - отметка книг как "прочитана" или "не прочитана"
- **Поиск** - поиск книг по ключевым словам в названии, авторе или описании
- **Рекомендации** - интеллектуальные рекомендации на основе избранных книг
- **Удаление** - удаление книг из библиотеки
- **Сохранение данных** - автоматическое сохранение в JSON файл и загрузка при запуске

## 🚀 Установка и запуск

1. Скачайте файл `library.py`
2. Убедитесь, что установлен Python 3.6 или выше
3. Запустите программу:
```bash
python library.py
